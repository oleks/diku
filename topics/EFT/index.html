Error-Free Transformations
An error-free transformation f is a binary operation which computes an approximation f'(x,y) of a mathematical function f(x,y) by transforming the input pair (x,y) into an output pair (r,e), where r corresponds to f'(x,y), and e corresponds to the absolute error of this approximation, i.e. f'(x,y) - f(x, y).
